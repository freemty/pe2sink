# Export Scope

This repository was created as a clean public export. It intentionally excludes:

- private git history from the development repository;
- model weights and full experiment outputs;
- paper source and submission files;
- private agent configuration, hooks, memory, and local machine paths;
- vendored third-party repositories.

The first public version keeps only the files required for:

- prompt-level comma-padding demonstration;
- compact paper-statistics figure reproduction;
- public smoke tests and minimal documentation.
